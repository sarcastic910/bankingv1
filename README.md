# 🏦 Banking System (Version 1)

A simple **Banking System** built in **C++** as a beginner Object-Oriented Programming (OOP) project.

This project was created to practice the fundamentals of C++ by building a menu-driven console application from scratch.

---

## 📌 Features

* Create a Bank Account
* Deposit Money
* Withdraw Money
* Check Account Balance
* Menu-driven interface using `switch` and `do-while`
* Basic input validation for deposits and withdrawals

---

## 🛠 Technologies Used

* C++
* Object-Oriented Programming (OOP)
* GCC (g++)
* Visual Studio Code
* Git & GitHub
